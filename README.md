# Repository's Organization

This repo became big quickly, so I made a lot of changes to make it as clean as possible, using lots of folders to properly separate my codes and knowledge:

## Platforms
The most important folder in this repo, it contains all of the problems I've made, separated by website, some of them, separated into their own categories (cof cof **CSES**)

Present platforms:
- Leetcode
- Codeforces
- CSES
- Atcoder
- Hackerrank
- Beecrowd

## Highlights
A more "learnful" folder where I keep not only my favorite/useful codes, but explanations and related problems about some key algorithms that i found cool to save.

## Contest

A temporary folder to store contest code, it should usually be empty, if it isn't, I got lazy or forgot to move the codes to their own folder in **platforms**, oops!

## Cemetery
A sad place where I keep failed problems from contests, I might solve them one day, or just leave them there, idk, time will tell...
